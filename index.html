UFPassPos

本项目为某系统的暴力破解.
由于该系统在某些重要功能下没有设置验证码,并直接显示用户的重要信息,导致不断利用POST请求进行暴力破解即可达到取得用户隐私的目的.
在 GitHub 中打开 UFPassPos

填入信息,开始破解

    // 学号(必填)
    String schoolId = "1205010101";

    // 生日(必填)
    final int startYear = 1993;
    final int startMonth = 07;
    final int startDay = 01;

    final int endYear = 1994;
    final int endMonth = 12;
    final int endDay = 31;

    // 性别(填写"男"或"女"，可不填)
    String sex = "男";

    // 填写身份证的省市信息(建议精确到城市，可不填)
    String province = "XX省";
    String city = "XX市";

利用测试类获取用户信息

本项目包括一些实用工具类

运行前请注意

请在eclipse中将该项目下每个文件的编码格式设置为"UTF-8",然后右键包含main函数的java文件, 选择Properties,在Run/Debug Settings中点击Edit,然后在Common窗口下将Encoding的值设置为GB2312, 执行完以上操作后运行程序,否则将出现中文乱码,导致项目无法正确运行.
其他开发工具请同样设置完成后再运行.

特别提醒

该项目仅用于演示,请使用自己的个人信息进行测试,严禁破解他人信息!
